# Cfn::Yats

AWS Cloudformation templates expressed with a Ruby DSL, YAML or JSON in a modular fashion

## Installation

Add this line to your application's Gemfile:

    gem 'aws-cfn-yats'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aws-cfn-yats
